# Hardware/KiCAD — Eagle-to-KiCAD port (work in progress)

This folder contains an in-progress port of the Margay v3.0 hardware design from Eagle to KiCAD.

**Current state:** footprint library (`Margay.pretty`), Eagle-import symbol library, and project/design-rule files are present. Schematic and PCB layout files are not yet complete.

See `Hardware/Margay_2/` for the most current working KiCAD design files (split-sheet schematic and PCB, also WIP).

Tracking issue: https://github.com/NorthernWidget/Project-Margay/issues/41
